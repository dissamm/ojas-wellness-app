# 🌸 Ojas Wellness App

AI-powered wellness application integrating menstrual cycle tracking, moon phases, and Ayurvedic Prakriti assessment.

## ✨ Features

- 🌙 **Cosmic Cycle Tracker** - Predict mood based on cycle phase + lunar phase
- 🔥 **Prakriti Assessment** - 18-question Ayurvedic body type quiz
- 🎵 **Curated Music** - Personalized recommendations based on your dosha
- 🧠 **AI Mood Prediction** - Trained on 5,659 real cycles
